## 🔍 자산 검색 및 분류 기준

### 1. 📂 저장소 명명 규칙 (Naming Convention)

모든 저장소 이름은 다음 형식을 따릅니다.

`[YYSS]-[CourseAcronym]-[TeamID]-[ProjectAcronym]`

| 구성 | 예시 | 설명 |
| :--- | :--- | :--- |
| **YYSS** | `24F` (2024 Fall) | 연도와 학기 (`S`: Spring / `F`: Fall) |
| **CourseAcronym** | `Capstone` (인공지능캡스톤디자인), `Solving` (인공지능문제해결) | 해당 과목의 약어 |
| **TeamID** | `T01` | 팀 이름 |
| **ProjectAcronym** | `SmartFarm` | 프로젝트의 약어 |
